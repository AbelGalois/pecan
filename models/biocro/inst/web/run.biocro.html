Run BioCro at a point

Usage

run.biocro(lat, lon, met.nc = met.nc, soil.nc = NULL, config = config, coppice.interval = 1)

Arguments

lat
latitude in decimal degrees
lon
longitude in decimal degrees
met.nc
full path and name of a netCDF file in PEcAn-CF format with meteorological driver data
soil.nc
full path and name of a netCDF file with soil data
config
full path and name of a config.xml file containing parameter values and configuration information for BioCro
coppice.interval
numeric, number of years between cuttings for coppice plant or perinneal grass (default 1)

Value

output from one of the BioCro::*.Gro functions (determined by config$genus), as data.table object

Description

Run BioCro at a point
