Write MODEL met files

Usage

met2model.BIOCRO(in.path, in.prefix, outfolder, overwrite = FALSE, ...)

Arguments

in.path
path on disk where CF file lives
in.prefix
prefix for each file
outfolder
location where model specific output is written.
...
can pass lat, lon, start.date and end.date

Value

OK if everything was succesful.

Description

Converts a met CF file to a model specific met file. The input files are calld /.YYYY.cf

Author

Rob Kooper, David LeBauer
