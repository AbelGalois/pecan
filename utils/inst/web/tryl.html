tryl

Usage

tryl(FUN)

Arguments

FUN
function to be evaluated for error

Value

FALSE if function returns error; else TRUE

Description

Test if function gives an error

Details

adaptation of try that returns a logical value (FALSE if error)

Examples

tryl(1+1)



[1] TRUE



# TRUE
tryl(sum("a"))



[1] FALSE



# FALSE
