Write sensitivity analysis configs

Usage

write.sa.configs(defaults, quantile.samples, settings, model, clean = FALSE, write.to.db = TRUE)

Arguments

quantile.samples
settings
list of settings
pft
pft id used to query PEcAn database
write.config
a model-specific function to write config files, e.g. write.config.ED
convert.samples
a model-specific function that transforms variables from units used in database to units used by model, e.g. convert.samples.ED
ensemble.samples
list of lists supplied by get.sa.samples

Value

list, containing $runs = data frame of runids, and $ensemble.id = the ensemble ID for these runs. Also writes sensitivity analysis configuration files as a side effect

Description

Write sensitivity analysis config files

Details

Writes config files for use in sensitivity analysis.

Author

David LeBauer, Carl Davidson
