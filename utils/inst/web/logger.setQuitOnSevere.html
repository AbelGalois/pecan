Configure whether severe should quit.

Usage

logger.setQuitOnSevere(severeQuits)

Arguments

severeQuits
should R quit on a severe error.

Description

The default is for a non-interactive session to quit. Setting this to false is especially useful for running tests when placed in inst/tests/test..R, but is not passed from tests/run.all.R.

Examples

## <strong>Not run</strong>: 
# logger.setQuitOnSevere(FALSE)
# ## <strong>End(Not run)</strong>
