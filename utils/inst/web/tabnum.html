Table numbers

Usage

tabnum(x, n = 3)

Arguments

x
numeric value or vector
n
number of significant figures

Value

x rounded to n significant figures

Description

Convert number to n significant digits

Examples

tabnum(1.2345)



[1] 1.23



tabnum(1.2345, n = 4)



[1] 1.234
