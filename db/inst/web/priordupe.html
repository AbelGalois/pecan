Duplicate PFT

Usage

priordupe(parent.pft.name = NULL, new.pft.name = NULL, new.pft.definition = NULL, settings = NULL)

Arguments

parent.pft.name
new.pft.name
new.pft.definition

Value

nothing, creates new pft in database as a side-effect

Description

Duplicate existing prior for new pft

Details

Creates a new pft that is a duplicate of an existing pft, including relationships with priors and species of the existing pft

Examples

## <strong>Not run</strong>: 
# priordupe(parent.pft.name    = "tempdecid",
#           new.pft.name       = "mytempdecid",
#           new.pft.definition = "mytempdecid is a new pft")
# ## <strong>End(Not run)</strong>
