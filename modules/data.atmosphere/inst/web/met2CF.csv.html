met2CF.csv

Usage

met2CF.csv(in.path, in.file, outfolder, format, lat = NULL, lon = NULL, nc_verbose = FALSE, ...)

Arguments

format
data frame or list with elements orig, bety, units for the original variable name, bety variable name, and original units. Columns with NA for bety variable name are dropped. Units for datetime field are the lubridate function that will be used to parse the date (e.g. ymd_hms or mdy_hm).
nc_verbose
logical: run ncvar_add in verbose mode?
