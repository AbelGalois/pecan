VPD

Usage

get.vpd(rh, temp)

Arguments

rh
relative humidity, in percent
temp
temperature, degrees celsius

Value

vpd: vapor pressure deficit, in mb

Description

Calculate VPD

Details

Calculate vapor pressure deficit from relative humidity and temperature.

Examples

temp <- -30:30
plot(temp, get.vpd(0, temp))
