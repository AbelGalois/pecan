Check convergence of multiple MCMC chains

Usage

check.convergence(mcmc.samples.list, threshold = 1.1, verbose = TRUE, autoburnin = FALSE, ...)

Arguments

mcmc.samples.list
mcmc.list object (from coda package) containing samples from MCMC chains.
threshold
Gelman-Rubin diagnostic parameter threshold. Default = 1.1
verbose
If TRUE, print convergence result. Default = TRUE
autoburnin
Whether or not to automaticall perform burnin in calculation of Gelman-Rubin diagnostic. Default = FALSE
...
Additional arguments to `gelman.diag` (from coda package)

Value

List length 3 containing the following:

convergence: Logical. Whether or not convergence was achieved.

diagnostic: Numerical value of the Gelman-Rubin multivariate diagnostic

error: Logical. Whether or not an error occured in the Gelman-Rubin calculation.

Details

Uses Gelman multivariate Gelman-Rubin diagnostic to check if multiple MCMC chains have converged
