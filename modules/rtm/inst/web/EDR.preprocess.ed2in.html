Preprocess ED2IN file for EDR

Usage

EDR.preprocess.ed2in(ed2in.path, output.path, config.path, datetime, history.full.prefix)

Arguments

ed2in.path
Path to original ED2IN file.
output.path
Path to directory for new ED2IN file (and where analysis is performed)
datetime
POSIX datetime object defining the time at which to run EDR
history.full.prefix
Full path and prefix for history file

Description

Copy ED2IN to `output.path` and modify for EDR specifications
