Spline Ensemble

Usage

spline.ensemble(gi.phii, median)

Arguments

gi.phii
matrix given as output from get.gi.phii
median
median value around which variance will be calculated

Description

Estimate model output based on univariate splines

Details

Accepts output from get.gi.phii (the matrix $g(\phi_i)$) and produces spline estimate of $f(phi)$ for use in estimating closure term associated with spline approximation
